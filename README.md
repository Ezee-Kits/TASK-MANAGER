# TASK-MANAGER

A modern Task Manager web application built with Django that allows users to create, update, organize, and delete tasks through a clean and intuitive interface.

## Features

* Add new tasks quickly and efficiently
* Update existing tasks
* Delete completed or unwanted tasks
* View all tasks in a structured table format
* Responsive and user-friendly design
* Django-powered backend with database integration
* Clean and maintainable code structure

## Technologies Used

* Python
* Django
* HTML5
* CSS3
* SQLite
* Bootstrap (if applicable)

## Purpose

This project was developed to demonstrate full-stack web development using Django, including CRUD (Create, Read, Update, Delete) operations, template rendering, database management, and deployment-ready project structure.
